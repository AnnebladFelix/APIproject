# How to!

## Install with cmd or terminal

### * Clone the repository
```
git clone https://github.com/AnnebladFelix/APIproject.git
```
### * Go into the repository
```
cd APIproject
```
### * Install dependencies
```
npm i
```

## Compile the project

### * Compile the typescript
```
tsc
```
### * Compile the Sass
``` 
sass src/scss:dist/css
```
